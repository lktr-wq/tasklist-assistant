# Day2 学习任务清单助手 Agent 指南

## 可用命令
- `python cli.py today`：查看今日全部学习任务。
- `python cli.py submit <任务编号> "<提交内容>"`：提交某个任务的完成说明。
- `python cli.py check <任务编号>`：查看某个任务的验收项和完成建议。

## 用户如何提问或调用
- 想知道今天要做什么时，运行 `python cli.py today`。
- 完成任务后，运行类似 `python cli.py submit 1 "完成阅读并整理笔记"`。
- 自查任务是否达标时，运行类似 `python cli.py check 1`。
- 如果不知道任务编号，先运行 `today`，再选择对应编号。

## 范围外拒答
- 如果用户要求查询未内置的课程、联网抓取资料或保存长期历史记录，应说明当前工具只覆盖 Day2 内置任务清单。
- 如果用户要求执行危险操作、删除文件或修改系统配置，应拒绝，并建议只运行本项目提供的三个安全命令。
- 如果用户要求把提交内容上传到外部服务，应说明本版本不连接外部 API。

## 工具失败处理
- 如果 Python 命令不可用，先尝试 `py cli.py today` 或检查 Python 是否已安装。
- 如果任务编号不存在，提示用户运行 `python cli.py today` 查看有效编号。
- 如果命令缺少参数，按帮助信息补齐参数后重试。
- 如果 git 推送失败，先检查远程地址、网络和 GitHub 登录状态，再重新执行推送。
