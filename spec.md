# Day2 学习任务清单助手 Spec

## 目标
- 提供一个零依赖 Python CLI，帮助学习者查看 Day2 今日任务。
- 支持 `today`、`submit`、`check` 三个命令的输入输出。
- 让学习者能围绕任务编号提交完成说明，并按验收项自查。

## 非目标
- 不实现用户登录、多人协作或权限系统。
- 不持久化保存提交记录，`submit` 只做本次命令的校验和确认输出。
- 不连接远程学习平台、数据库或外部 API。

## 边界条件
- 当任务编号不存在时，输出清晰错误，并提示使用 `today` 查看任务清单。
- 当 `submit` 的提交内容为空时，拒绝提交并返回错误。
- 当命令缺少参数或命令未知时，展示 argparse 提供的用法提示。

## 验收标准
- 运行 `python cli.py today` 能看到至少 3 个 Day2 任务及其验收项。
- 运行 `python cli.py submit 1 "完成阅读并整理笔记"` 能看到提交成功、任务标题和提交内容。
- 运行 `python cli.py check 1` 能看到任务 1 的验收项、完成建议和检查结果。
- 运行 `python cli.py check 999` 返回不存在该任务的错误信息。
